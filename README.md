Software Engineer focused on building developer tools, AI agents, and production-grade systems with Rust, Python, TypeScript, and Go. <br><br>

I work at the intersection of systems engineering, backend infrastructure, and applied AI — especially on tools that help developers move faster while keeping reliability, security, and maintainability in mind.

Currently, I’m building Navi, an open-source coding agent engine written in Rust and designed as part of a broader developer infrastructure toolkit. Navi explores multi-provider LLM orchestration, streaming execution, auditable security policies, WASM-based extensibility, virtual file systems, persistence, observability, and developer workflows built for real-world usage rather than demos.

My technical interests include developer tools, AI agents, Rust systems programming, backend infrastructure, cloud-native services, observability, production reliability, and search/indexing systems.

I enjoy working on complex technical problems, debugging difficult systems, optimizing performance, and turning ideas into useful software. I care about choosing the right tool for the problem and building systems that are understandable, maintainable, and practical to operate.

Education: Software Engineering — Federal Institute of Pernambuco (IFPE), Brazil.

Open to Software Engineer opportunities in developer tools, AI-powered products, backend infrastructure, cloud platforms, and systems software.

My work is driven by a powerful concept: **Ikigai (生き甲斐)**.

> **Ikigai is a Japanese term for "a reason for being."** For me, I've found it in the art of software development.
>
> Software development is my Ikigai. It's what I love to do.
